# Updates Log

Use this folder to track ongoing changes to the knowledge base.

## Structure

- `daily/` - short day-to-day notes
- `weekly/` - weekly summaries and decisions

## Recommended Format

- Date-first filenames: `YYYY-MM-DD-short-title.md`
- Keep each entry concise
- Include links to updated docs and related issues/PRs
